Math Test
Задание
Необходимо реализовать игровую логику и код для ее проверки.
Используемые средства:
•	Java SE 1.8
•	Maven with modelVersion=4.0.0
Файлы: Test Slot Game - math model.xlsx
Сроки выполнения: одна неделя
Общие указания
	В данном задании необходимо реализовать слот машину с игровым полем 3 на 3. В игре заданы 5 линий выплат. 
	Reference — https://betsoft.com/play-demo/?id=5707. Обратите внимание на правила, приведенные для этой игры. 
	В них подробно описаны механики игры. В данном задании нужно реализовать 
	лишь базовую часть референсной игры с некоторыми модификациями, введенными для упрощения.


Описание
	Игрок делает ставку = 5 кредитов. Машина крутит барабаны (англ. Reels) и останавливает их на случайной позиции 
	(позиция остановки для каждого барабана своя).

	Машина анализирует выпавшие иконки на линиях выплат. Если на линии выпадают иконки, 
	составляющие какую-либо выигрышную комбинацию, игрок получает выплату, соответствующую таблице выплат.

	Написанное приложение должно содержать реализацию игровой логики и код для проверки возвратности (RTP) игры. 
	Проверка заключается в разыгрывании N раундов и проверкой практической возвратности игры.
	
	Программа должна принимать на вход количество тестовых итераций и выводить следующую информацию по окончании работы:
•	сумма выигрыша (Total Outcome);
•	частота выигрыша (Total Hit Frequency)
•	возвратность игры (Return to Player).


Результат
Результатом выполненной работы должны быть:
•	Исходный код классов с игровой логикой;
•	Тестовый класс с математическим тестом игровой логики;
•	Unit тесты для покрытия игровой логики.


Пример работы программы
Входное значение
2000
Выходное значение
Total Bet: 10000
Total Win: 5092
Total Hit Rate 45.4%
Return to Player (RTP): 50.92%
